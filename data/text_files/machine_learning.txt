
Introduction to Machine Learning

Machine Learning is a branch of Artificial Intelligence that enables computers to learn patterns from data
without being explicitly programmed.

Types of Machine Learning:
1. Supervised Learning
2. Unsupervised Learning
3. Reinforcement Learning

Common Algorithms:
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- K-Means Clustering

Applications include recommendation systems, image recognition, spam detection, and predictive analytics.
